Memory Management Simulator is a C++ based Operating System project that demonstrates core memory management concepts including Paging, Segmentation, Virtual Memory, Page Replacement Algorithms, Memory Allocation Strategies, and Performance Analysis.

The project provides a menu-driven interface to simulate memory management techniques used in modern operating systems.

---

## Features

### Paging

* Logical to Physical Address Translation
* Page Number Calculation
* Offset Calculation

### Page Replacement Algorithms

* FIFO
* LRU
* Optimal
* Belady's Anomaly Demonstration

### Memory Allocation

* First Fit
* Best Fit
* Worst Fit
* Allocation Comparison

### Segmentation

* Segment Table
* Base and Limit Registers
* Address Translation
* Segmentation Fault Detection

### Virtual Memory

* TLB Simulation
* Page Table Simulation
* Page Fault Handling
* Disk Simulation
* Statistics Tracking

### Performance Dashboard

* Algorithm Comparison
* Page Fault Analysis

### Memory Visualization

* Memory Layout Visualization
* Frame Visualization
